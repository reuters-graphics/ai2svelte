**ai2svelte** is Reuters Graphics' custom adaptation of [ai2html](http://ai2html.org/), the popular open-source script originally developed by The New York Times.

This powerful tool streamlines your workflow by automatically converting Adobe Illustrator artwork into clean, responsive Svelte components, making it easy to integrate your designs directly into web applications.

---

### Installation

Download the latest `.zxp` file from our releases and use the ZXP Installer to install the plugin.

<CardGrid>
  <LinkCard
    title="ZXP Installer"
    href="https://aescripts.com/learn/zxp-installer/"
  />
  <LinkCard
    title="Latest Releases"
    href="https://github.com/reuters-graphics/ai2svelte/blob/master/ai2svelte/dist/zxp/com.reuters-graphics.ai2svelte.zxp"
  />
</CardGrid>

Once installed, the plugin can be accessed via the `Window` > `Extensions` menu in Adobe Illustrator.

<Aside>
  If open, restart Adobe Illustrator after installation to activate the plugin.
</Aside>
