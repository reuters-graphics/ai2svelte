**ai2svelte** is Reuters Graphics' custom adaptation of [ai2html](http://ai2html.org/), the popular open-source script originally developed by The New York Times.

This powerful tool streamlines your workflow by automatically converting Adobe Illustrator artwork into clean, responsive Svelte components, making it easy to integrate your designs directly into web applications.

## Installation

Choose from multiple installation methods to get ai2svelte up and running in your Adobe Illustrator workflow.

### Adobe Exchange

Install directly from Adobe's official marketplace for the most streamlined experience.

<LinkCard
  title="Adobe Exchange"
  href="https://exchange.adobe.com/apps/browse/cc?product=All&listingType=plugins&sort=MOST_RECENT"
/>

### ZXP Installer

For manual installation, download the latest ZXP file from our releases and use the ZXP Installer.

<CardGrid>
  <LinkCard
    title="ZXP Installer"
    href="https://aescripts.com/learn/zxp-installer/"
  />
  <LinkCard
    title="Latest Releases"
    href="https://github.com/reuters-graphics/ai2svelte/ai2svelte/dist/zxp/com.reuters-graphics.ai2svelte.zxp"
  />
</CardGrid>

The plugin can now be accessed from the `Window` > `Extensions` menu in Adobe Illustrator.

<Aside>
  If open, restart Adobe Illustrator after installation to activate the plugin.
</Aside>
